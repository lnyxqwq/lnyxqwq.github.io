题意：

给定长为 n 的序列 a，求其所有长度为奇数的子区间的中位数之和

若答案为 A​A​A​，你的答案为 B​B​B​，则只需保证 ⌊A2⌋≤B≤2A​\lfloor \frac{A}{2} \rfloor \le B \le 2A​⌊2A​⌋≤B≤2A​ 即可通过

1≤n≤1e5,1≤ai≤1e181 \le n \le 1e5,1 \le a_i \le 1e181≤n≤1e5,1≤ai​≤1e18

😅😅😅

首先有一个 O(n2log⁡n)O(n^2\log n)O(n2logn) 的做法，首先枚举中位数 xxx，然后将大于等于 xxx 设为 111 ，小于的设为 000 ，然后拿两个 bitbitbit 搞一搞可以求出中位数大于等于 xxx 的奇数区间个数，容斥一下就能求出等于 xxx 的了

但是现在给了你一个性质，你需要用这个性质把他优化掉一个 nnn ，发现这个 bitbitbit 的 nnn 根本不能动，所以就只剩下值域的 nnn 了，我们把所有数变成小于等于他的二的整次幂，发现这样变化后一定大于原数一半，这样就做完了，时间复杂度 O(nlog⁡2n)O(n\log^2 n)O(nlog2n)

优化：

  1. 通过这个在下面说的东西优化掉一个 log⁡\loglog ，现在时间复杂度是 O(nlog⁡n)O(n \log n)O(nlogn)

  2. 发现我求出来的答案是 ⌊A2⌋≤B≤A\lfloor \frac{A}{2} \rfloor \le B \le A⌊2A​⌋≤B≤A 区间的，有一半没用，所以可以把所有数变成小于等于他四的整次幂，然后把答案乘 222
